# Raytracer Project
A small c++ based raytracing engine project to learn about how raytracing works. 
"Raytracing in One Weekend" by Peter Shirley used as a guide.

---

### Helpful info:
- to run:
delete build folder
run "cmake -B build"
run "cmake --build build"
run "build/Debug/InOneWeekend.exe > image.ppm"
view image.ppm with photoshop or other ppm viewer

- [link to the guide](https://raytracing.github.io/books/RayTracingInOneWeekend.html#metal/adatastructuretodescriberay-objectintersections)
